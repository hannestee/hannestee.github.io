Assignment: Graphic Web Designer
Fenno Optiikka

1.8.2017-1.6.2018

The purpose of the project was to revamp the website of the Finnish optician chain Fenno Optiikka. Key feature was to add a gallery of available glasses for the public to see as the website at that point lacked it. In the project we worked in a team of ~5 people so it was a small scale project.

  • As a Graphic Web Designer I designed website layouts with Photoshop and then sent them to the Wordpress expert in our team.
  • I communicated with the client both face-to-face and remotely on the new features of the site and how the content should be distributed to each page.
  • After the Wordpress site was coded I inspected it for errors both in functionality and the appearance. I also received feedback from the client and translated it to rest of the team.
  • I acted as a kind of mediator in determining which requested features could be added realistically in the time frame which was given from the management. Sometimes compromises were made.
  • I managed the content itself on the Wordpress site and made changes according to the clientâ€™s requests. Aside from the content I did also take a look under the hood of Wordpress and while learning the code I did some minor changes.
  • Tools & Technologies used: Photoshop, Wordpress, PHP, CSS, Skype
